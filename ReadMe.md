# musinsa_homework_강지현

### Description

- 무신사 인게이지먼트팀 지원 과제입니다. 시간 분배 착오로 인하여 급하게 작업하여 미완성 결과이며(테스트코드 및 벨리데이션 누락) 잔버그 이슈가 있는점 참고부탁드리겠습니다.

  <br/><br/>


### 참고사항

- h2.console 확인 시 파일경로가 아닌 console에 h2-console/ 이하 url로 접속가능합니다.
- 서비스에 각 메소드에 간단히 주석을 기재하였습니다.
### Stack
    Spring-boot
    JPA
    hibernate
    MySQL
